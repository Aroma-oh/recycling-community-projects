## 📢 쓰쓰또쓰를 소개합니다. 
**"잘" 분리 배출된 플라스틱 비닐의 8%만이 재활용된다는 사실을 아시나요?** <br><br>
쓰쓰또쓰는 낮은 재활용률의 한계를 탈피하고자 사용자에게 버리지 말고 더 쓸 것을 제안 합니다. <br>
사용자가 두번, 세번 더 쓴다면 단번에 재활용률 100%, 200%를 달성한 것이지 않을까요? 👀 <br>
